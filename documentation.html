Design Process

Features

The design proess for this chart was more challenging than I intitially expected. I wanted to make a chart that would show the price of the Nvidia stock over time, but a simple chart plotting that proved to have many problems. Then I changed the elements of the chart. Originally it was just a chart of the closing price over time, but I decided to make it into a candlestick chart instead, which is a type of chart used to show opening and closing prices on each day. This also didn't work because I found it was too cluttered and hard to interpret. So I instead decided to make an interactive chart that allows the user to look at any aspect of the dataset they want The new chart was a line chart of the closing price over time by default, but using a dropdown the user could choose to look at opening price, the high and low values, and the volume of shares traded on that day. This was much better, as it allows the user to look at the information of their choosing. This also had problems because the stock value grew exponentially over time, so the earlier years were impossible to distinguish as a result. To fix this I made the y axis scale logarithmically, to better show the increases over time. This didn't work for everything, for example volume, so I decided to add a drop down that allows the user to choose between logarithmic and linear scaling for the y axis. The final feature I added was a year select feature. This let the user select a starting and ending year for the chart, so they can look at more specific time periods and compare them if they want to.

Design Choices

When I finished making the chart, it was very ugly. It was just a black line on a white background with unlabeled axes. The first thing I did to fix this was add labels to the axes to make this easier to understand. Then, I changed the color of the line in the chart to be lime green. This is because lime green is the primary color of Nvidia, and I thought it would be fitting for the chart to subtly convey that. I also changed the background of the chart to a grey, as grey is a main color of Nvidia as well. I thought about changing the font but I didn't like any of the font options and the more I think about it the better I think the default font looks.
